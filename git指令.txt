1.初始化 Git Repository        
  -- git init

2.觀察 Repository 檔案追蹤狀況 
  -- git status

3.將檔案加入追蹤(index)清單    
  -- git add 檔名  (單一檔案)
  -- git add .  (所有檔案)

4.建立一組版本更新(Commitment) 
  -- git commit -m "版本更新訊息"

5.查看分支(Branch)
  -- git branch
  新增分之
  -- git branch cat (新增 cat分之)
  分之改名
  -- git branch -m cat tiger(cat 改名 tiger)
  刪除分之
  -- git branch -d cat
  切換分之
  -- git checkout cat
  新增分之並切換
  -- git checkout -b cat
  從github 下載分之
  --git checkout -b newlocalbranchname origin/branch-name
  

6.查看版本紀錄
  -- git log

7.查看遠端的Repository
  -- git remote -v

8.加入遠端的Repository (只要做一次)
  -- git remote add 遠端空間的名稱(自取) 網址

9.第一次下載 GitHub 雲端專案
  -- git clone 遠端空間的網址 本機資料夾名稱(可以省)

10. 從GitHub 雲端專案下載合併跟新
  -- git pull 遠端空間的名稱 遠端空間的分支
